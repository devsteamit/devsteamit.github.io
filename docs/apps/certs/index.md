# AWS Certification Practice Apps

This documentation describes the family of mobile applications used to prepare for AWS certification exams.

Each app focuses on a specific AWS certification.

---

| Certification                         | Documentation     |
|---------------------------------------|-------------------|
| AI - Practitioner                     | [AIF-C01](aifc01) |
| Cloud - Practitioner                  | [CLF-C02](clfc02) |
| Data Engineer - Associate             | [DEA-C01](deac01) |
| Machine Learning Engineer - Associate | [MLA-C01](mlac01) |
| Solutions Architect - Associate       | [SAA-C03](saac03) |
| Developer - Associate                 | [DVA-C02](dvac02) |
| Solutions Architect - Professional    | [SAP-C02](sapc02) |
| DevOps Engineer - Professional        | [DOP-C02](dopc02) |
| Machine Learning - Specialty          | [MLS-C01](mlsc01) |
| Advanced Networking - Specialty       | [ANS-C01](ansc01) |
| Security - Specialty                  | [SCS-C03](scsc03) |

---

# How the Apps Work

All applications share the same learning engine but contain different sets of exam questions.

Main features:

• Practice exams  
• Study mode  
• Progress tracking  
• Offline functionality
